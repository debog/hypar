# hypar documentation

Welcome to the hypar documentation.

This documentation is built with Sphinx and MyST (Markdown) and hosted on Read the Docs.

Navigation:
- [Installation](installation.md)
- [Usage](usage.md)
- [API](api.md)

Getting started
- If you'd like to preview the documentation locally:
  1. Install the docs requirements: `pip install -r docs/requirements-docs.txt`
  2. From the `docs/` directory run: `make html`
  3. Open `_build/html/index.html` in your browser

Contributing
- Add or update pages inside the `docs/` folder.
- For API documentation, we use Sphinx autodoc. See `api.md` for instructions.
